# MediaDock – Home Assistant Add-on

Self-hosted personal media collection manager. Track vinyl records, CDs, games,
Blu-rays, DVDs, books and more – with automatic metadata from MusicBrainz and
cover art from the Cover Art Archive.

## How to use

1. Install the add-on from your local repository (see the repository README).
2. Start the add-on.
3. Click **Open Web UI** or enable **Show in sidebar**.
4. Hit **+ Hinzufügen** to add your first item.

## Configuration

This add-on has no user-configurable options. All data is stored automatically
in `/data/collection.db`.
